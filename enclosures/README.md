# Enclosures

3D print files for the project. No supports needed in any of the files.

For the most up-to-date designs, see Tinkercad profile [TinkerGui](https://www.tinkercad.com/users/7svhW7zioAZ).

## Pico case

Files: `Raspberry Pi Pico 2 W case - bottom.stl`, `Raspberry Pi Pico 2 W case - top.stl`

Based on maximilliampere's [slim modular Pico case](https://www.printables.com/model/61680-slim-modular-raspberry-pi-pico-case).

Print settings:
- Layer height: 0.2 mm
- Infill: 20%

## Sign box

Files: `On Air box - base.stl`, `On Air box - lid.stl`, `On Air box - letters.stl`

Houses the 12-LED WS2812 NeoPixel ring. Custom designs.

Print settings (base):
- Layer height: 0.2 mm
- Infill: 15%

Print settings (lid and letters):
- Layer height: 0.3 mm
- Infill: 15%
